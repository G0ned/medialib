Medialib is a web application aiming to help gamers to organize and manage their very own multiplatform videogame library.

## Description

The existence of various videogame platforms and stores within the same platform makes it tedious to have a control of the purchased games as well as yout wishlist. Medialib tries to unify yout collection of videogames on one unique platform so it is easy for you to have a complete collection of your purchased games. 

## Characteristics

- User registration and login
- Use of an external API with progressive cache in a local database
- Search for videogames
- Create collections of videogames

## Technologies

- **Backend:** Python 3.11, Django 5.2, Django REST framework
- **Database:** PostgreSQL
- **Tests:** pytest
